Can you think of an analogy to describe HTML, CSS and the DOM?

HTML (Hyper text markup language) is like the walls, framing Etc
CSS (Cascading Sytle Sheets) define the colours, decorations, sytles, fonts etc.
The DOM (Document Object Model) is like the convention for combining the HTML and CSS together. Eg. All walls are coloured beige. All window sills are coloured white etc.
